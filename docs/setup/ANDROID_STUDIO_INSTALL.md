# Android Studio Installatiehandleiding

Android Studio is nodig voor de ontwikkeling en het runnen van de Android frontend.

## Installatie stappen

1. Ga naar de [Android Studio Download pagina](https://developer.android.com/studio).
2. Download de nieuwste versie van Android Studio (Ladybug of nieuwer).
3. Volg de installatie-wizard voor jouw besturingssysteem.
4. Kies bij de eerste keer opstarten voor de **Standard** installatie.
5. Zorg ervoor dat de **Android SDK**, **Android SDK Platform** en **Android Virtual Device (AVD)** zijn geselecteerd voor installatie.

## Configuratie voor dit project

1. Open dit project in Android Studio via `File > Open`.
2. Ga naar `Settings` (of `Preferences` op Mac) > `Build, Execution, Deployment` > `Build Tools` > `Gradle`.
3. Zorg dat de **Gradle JDK** is ingesteld op **JDK 21**.
4. Maak een emulator aan via de **Device Manager** (API level 28+ aanbevolen).
